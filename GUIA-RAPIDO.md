# ⚡ GUIA RÁPIDO - INT Tools

## 🚀 SETUP NO OUTRO PC (Cole no terminal)

```bash
# 1. Clonar projeto
git clone https://github.com/Kaesene/int-tools.git
cd int-tools

# 2. Instalar dependências
npm install

# 3. Rodar projeto
npm run dev

# 4. Abrir no navegador
# http://localhost:3000
```

---

## 📝 COMANDOS MAIS USADOS

```bash
# Ver status do Git
git status

# Baixar atualizações
git pull

# Adicionar mudanças
git add .

# Fazer commit
git commit -m "sua mensagem"

# Enviar para GitHub
git push

# Rodar projeto
npm run dev

# Build de produção
npm run build
```
