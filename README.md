# 🛠️ INT Tools - E-commerce de Ferramentas e Tecnologia

Loja virtual moderna e profissional especializada em **ferramentas** e **tecnologia importada** de alta qualidade.

## 🎨 Design

- **Identidade Visual:** Mix Tech + Industrial
- **Cores Primárias:**
  - Azul Tech: `#0066FF`
  - Laranja Industrial: `#FF6B35`
- **Layout:** Totalmente responsivo (Mobile, Tablet, Desktop)

## ✨ Funcionalidades (Fase 1 - MVP)

- ✅ Homepage com Hero Section impactante
- ✅ Grid de produtos em destaque
- ✅ Sistema de categorias visuais
- ✅ Header responsivo com carrinho
- ✅ Footer completo com links e redes sociais
- ✅ Cards de produtos com avaliações e preços
- ✅ Design moderno com animações suaves

## 🚀 Tecnologias

- **Framework:** Next.js 16 (App Router)
- **Linguagem:** TypeScript
- **Estilização:** Tailwind CSS 4
- **Deploy:** Vercel
- **Versionamento:** Git + GitHub

## 📦 Instalação

```bash
# Clonar repositório
git clone https://github.com/Kaesene/int-tools.git

# Entrar na pasta
cd int-tools

# Instalar dependências
npm install

# Iniciar servidor de desenvolvimento
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no navegador.

## 📂 Estrutura do Projeto
